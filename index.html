HTML

  • The head element contains information about a webpage
  • The body element represents the visible content shown to the user
  • HTML uses 6 levels of heading elements. h1 - h6
  • Footer commonly used for author, copyright, contact information, or other related links/info

CSS

  • CSS is used for styling an HTML page
  • CSS uses the box model to determine how things are displayed
  • A CSS sheet must be linked to style a page
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

GIT

  • GIT can be used by a team to collaborate on projects
  • GIT is used to track changes made and can revert changes
  • GIT allows you to work on code without impacting the live server

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • JavaScript can dynamically change a webpage
  • JavaScript allows websites to be interactive
  • JavaScript can be used to create objects, and functions to give life to a page
